Intro to App Development Workshop
Calling all app developers! We'll be hosting an introduction to app development workshop. This will be self-paced with an expert mentor to help out with questions you may have. We support Android and iOS apps. Before you get started, we recommend downloading the required IDEs. iOS apps require a Mac computer and installing XCode through the app store. Android apps can be developed on any type of computer and require Android Studio to be installed. If you haven't started app development yet, we recommend following either of these tutorials:
Android: Beginning Android Development Tutorial: Introduction to Android Studio
iOS: iOS 10 App Development Essentials 1: Create Your First App

Intro to GitHub Workshop
Have you ever worked on a coding project with a team, but have run into trouble sharing code between you? GitHub is an extremely useful and powerful versioning software tool that helps teammates share and update programs between each other. In this workshop, you'll learn how you can share code between you and your teammates. If you haven't already done so, make sure to create an account GitHub's website.
